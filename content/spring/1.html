Spring Boot Introduction

What is the use of Spring Boot?

Spring Boot is an extensionof the spring framework which eliminates boilerplate configutrations required for developing/settingup a spring application.

Spring Boot is very useful for developing web applications by reducing code length.

Spring Boot is the way to develop Spring based applications rapidly by avoiding manual configutrations

Entry Point (Starting point) of Spring Boot Application Execution

Entry point of Spring Boot Application is a class which annotated with the @SpringBootApplication

Basic Maven Dependencies in Spring Boot Application

Spring Boot Dependencies are different than Spring Boot Dependencies. Spring Boot Dependencies are very easy to use with adding of less dependencies compared to Spring framework.
