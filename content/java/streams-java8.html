Streams in java-8

What is streaming

Streaming is the process of processing a data, for example loading video from server is an example for streaming.

Remmember Streams never stores the data in memory like data structors

What is stream pipeline

Stream pipeline is a sequence of operations applying on Streams, where streams has methods to operate on data sources like Collections and Arrays

Steam pipeline is a aggregation of operations which returns again stream only.

Stream pipeline can be done with intermediate operations and terminal operations

components of pipeline

Data Source : on which stream operations are being applied after converting into stream

Intermediate methods: these are the methods wchich will be performed on the stream and returns updated stream as a result. we can apply sequence of methods too

Terminal methods: these are the methods which will be applied on final stream after applying intermediate operations. for example forEach()
