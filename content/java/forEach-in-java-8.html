forEach in java-8

What is forEach() method

forEach() method available from Iterable interface, to iterate Collections and Streams. forEach() method is avaialable from java-8 version.

forEach() method required action object of Consumer class

forEach() method is a terminal operation method, that means which will be invoked as a last method always.

We can create action object of Consumer calss in two different ways. one is creating a class which implements Consumer interface by overriding action() method, and second way is by using Lambda funtions.

We can call/invoke forEach() on collection object or stream object by passing action object as a parameter or directly by passing lambda funtion as an action
