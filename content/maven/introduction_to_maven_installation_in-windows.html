Introduction to maven and installing maven in windows

What is maven

Maven is a build tool for Java based applications. Maven will add required dependencies or libraries to your application class path and build final executabale or production ready application of your developed code.

pom.xml is a file which contains details about project dependencies.

How to install maven in windows

You must install java first, if java not installed in your system. Then you have to made JAVA_HOME environament variable pointing to JDK folder location.

If your system installed with Java then we can install maven in your system.
